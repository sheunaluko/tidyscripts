# TOBI - Tidyscripts Observability Interface

An observability dashboard for monitoring and analyzing tidyscripts applications.

## Features

### Metrics Dashboard
- Real-time CPU usage monitoring
- Memory consumption tracking
- Request rate visualization
- Time-series data display with mini charts

### Logs Viewer
- Centralized log aggregation
- Real-time log streaming
- Searchable and filterable logs

### Health Monitoring
- Service health checks
- Component status tracking
- System availability monitoring

## Usage

Navigate to `/laboratory/tobi` to access the observability interface.
