# Function: bar\_chart()

> **bar\_chart**(`data`): `void`

Creates a bar chart from specified data. 
Data is an array of arrays, of the format shown below: 
``` 
var test_bar_data = [
  ["Fruit" , "Value" ] ,
  ["Apple" , 1 ] , 
  ["Banana" , 2] ,
  ["Pear" , 1 ] , 
]
```
